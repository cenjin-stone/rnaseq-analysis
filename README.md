# RNA-seq 差异表达分析流程

## 项目简介
本项目使用 bulk RNA-seq 数据（人类肺癌细胞系 H520，GSE237935），分析**处理组 vs 对照组**的转录组变化。共 4 个样本（2 对照 + 2 处理）。

## 注意事项
由于本地计算资源限制（内存不足），本分析仅使用 **1号染色体** ，对 **4个样本** 进行流程验证。  
结果图中的表达量仅反映 1号染色体上的基因，**不代表全基因组表达情况**，仅供代码测试使用。

## 分析流程
1. **数据清洗**：fastp（去接头、低质量过滤）
2. **比对**：STAR（1号染色体参考基因组）
3. **定量**：featureCounts
4. **差异分析**：DESeq2（padj < 0.05，|log2FC| > 1）
5. **可视化**：火山图、热图（ggplot2、pheatmap）

## 文件说明
- `R-scripts/`：R 语言分析脚本
- `shell-scripts/`：Linux 命令行脚本
- `counts_matrix.txt`：表达矩阵（行=基因，列=样本）
- `volcano.png` / `heatmap.png`：结果图

## 结果
- 差异基因数：30 个（5 上调，25 下调）
- 火山图：红色为上调，蓝色为下调

## 🔬 环境
- Linux: CentOS 7
- 主要工具：fastp v1.1.0, STAR v2.7.11b, featureCounts v2.1.1, DESeq2
